# **App Name**: Mystic Forge AI

## Core Features:

- Personalized Input Form: A visually engaging form for users to enter personal details (e.g., name, birth month, intention, mood, interests) that will inform their blessing.
- AI Oracle Blessing Generation: A generative AI tool that crafts mystical and poetic blessings and prophecies by transforming user input into symbolic archetypes for an LLM.
- Ritualized Fortune Revelation: An immersive, animated front-end sequence featuring particle effects, glowing symbols, and gradual text manifestation to reveal the generated fortune.
- User Account Management: Enable user registration and login functionality using Firebase Authentication to personalize the experience.
- Personal Fortune History: Securely store generated blessings in Firestore and allow users to revisit and view their past oracle readings.
- Shareable Blessing: Provide an option to easily share the generated mystical blessings as an image or text across various social platforms.

## Style Guidelines:

- Dark color scheme to evoke a mystical and ancient oracle atmosphere.
- Primary color: A deep, ethereal violet (#6826D9), representing mystery and wisdom.
- Background color: A muted purplish-grey (#26202B), creating a contemplative and immersive backdrop.
- Accent color: A vibrant blue-violet (#3D70F5) for subtle glow effects and interactive elements, analogous to the primary hue.
- Headline font: 'Playfair' (serif) for elegant, high-contrast titles that enhance the mystical and high-end feel.
- Body text font: 'PT Sans' (sans-serif) for clear readability of blessings and general content, complementing the elegant headlines.
- Use symbolic and arcane icons for interactive elements, such as glowing runes, celestial motifs, or simplified tarot card designs.
- Focused, ritualistic layout design emphasizing central elements and progressive reveals, with screen darkening and atmospheric transitions during fortune generation.
- Fluid, magical animations for user interaction and fortune revelation, including floating particles, glowing symbols, gradual text materialization, and subtle background environmental effects like drifting smoke or a faint celestial swirl.
